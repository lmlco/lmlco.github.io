Works

03 / Case
Recruitment film

応募前の不安を消す、採用・企業PR映像

働く人の表情と現場の温度を短尺に。

→
ヒアリングで拾った「入社の決め手」を軸に構成台本を設計。本編に加え縦型ショートへ派生させ、採用ページ・SNS・説明会資料で使い回せる資産として納品しました。
Photo asset

採用に効く撮り下ろし写真

Web・SNS・資料で使い回せる資産に。

→
現場の空気感を残しつつ、トリミング耐性の高い構図で撮影。採用サイト・会社紹介資料・SNS投稿の3用途を1回のロケでまかなえるカット構成にしました。
MEO / Local

地図で選ばれる店舗導線（MEO）

写真・紹介文・投稿を整え、来店前に安心を。

→
Googleビジネスプロフィールの写真・紹介文・投稿頻度を一括整備。来店前の比較検討で選ばれやすい情報設計に組み直しました。
Short video ops

短尺動画を“続く仕組み”にする

継続投稿のフォーマットと編集テンプレを構築。

→
1回の撮影から複数フォーマットを量産できる編集テンプレを設計。ネタ出し〜編集〜投稿までのサイクルを型化し、担当者が変わっても止まらない運用にしました。
Production support

HP・ポスター・サイネージまで一気通貫の制作物支援

媒体ごとの見た目を、ひとつのトーンへ整理。

→
会社サイト・採用ポスター・店頭サイネージ・業務ソフトのUIまで、バラバラに発注されていた制作物を同じトーン＆マナーで整理。制作コストと発注の手間を抑えました。
LINE Biz

予約・問い合わせをLINEで完結させる導入支援

電話・メールの導線をLINE公式アカウントに一本化。

→
予約受付・よくある質問の自動応答・顧客管理をLINE公式アカウントに統合。電話対応の負荷を下げつつ、来店・再来店のフォローが自動で回る仕組みを整えました。
